# 🤖 AI Resume Screening System

An AI-powered Resume Screening System that matches resumes with job descriptions using NLP and Machine Learning techniques.

This project uses:

- TF-IDF Vectorizer
- Cosine Similarity
- Natural Language Processing (NLP)

---

# 🚀 Project Overview

The goal of this project is to automate resume screening and candidate shortlisting.

The system compares resumes with job descriptions and calculates similarity scores to identify the best matching candidate.
